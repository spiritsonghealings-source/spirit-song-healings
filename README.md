# Spirit Song Healings

GitHub-ready Vite + React site for GitHub Pages.

## Install
```bash
npm install
```

## Run locally
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Before publishing
Replace:
- `https://calendly.com/your-link`
- placeholder testimonials
- your contact details
- any wording you want changed

If your repo name is different, update the `base` value in `vite.config.js`.
